## 项目概述

研究玻璃期货的价格规律与量化交易策略，探索趋势跟踪和均值回归策略在商品期货市场上的表现。

## 数据范围

- **品种**：郑州商品交易所玻璃期货（FG0 主力连续合约）
- **来源**：[AKShare](https://www.akshare.xyz/)
- **时间范围**：2012-12 至 2026-06
- **数据维度**：开高低收、成交量、持仓量

## 研究内容

### 1. 技术指标

- 双均线系统（MA10 / MA30）
- MACD 趋势指标
- RSI 超买超卖
- 布林带均值回归

### 2. 回测策略

| 策略 | 总收益 | 最大回撤 | 说明 |
|------|--------|---------|------|
| 买入持有 | -25.17% | - | 玻璃期货长期下跌趋势 |
| 双均线交叉 | -38.74% | -71.17% | 趋势跟踪，在震荡市表现差 |
| RSI 均值回归 | - | - | 超买做多、超卖做空 |
| 布林带反转 | - | - | 触及上下轨反向交易 |

### 3. 初步结论

从 2012 年至今的回测结果来看：

- 玻璃期货长期处于震荡下行通道
- 简单双均线策略在该品种上表现不佳，主要因为假突破和震荡损耗
- RSI 和布林带均值回归策略阶段性有效，但需要配合止损和仓位管理
- 未来研究方向：加入成交量过滤、波动率阈值、以及多品种配对交易
