## 策略原理

双均线策略是最经典的技术分析策略之一：

- **金叉**：短期均线上穿长期均线 → 买入信号
- **死叉**：短期均线下穿长期均线 → 卖出信号

## Backtrader 实现

```python
import backtrader as bt

class DualMAStrategy(bt.Strategy):
    params = (
        ('short_period', 10),
        ('long_period', 30),
    )
    
    def __init__(self):
        self.short_ma = bt.indicators.SMA(period=self.p.short_period)
        self.long_ma = bt.indicators.SMA(period=self.p.long_period)
        self.crossover = bt.indicators.CrossOver(self.short_ma, self.long_ma)
    
    def next(self):
        if not self.position:
            if self.crossover > 0:
                self.buy()
        elif self.crossover < 0:
            self.sell()
```

## 参数优化

使用 `optunity` 或 Backtrader 内置的 `Cerebro.optstrategy` 进行参数扫描：

| 短周期 | 长周期 | 年化收益 | 最大回撤 |
|--------|--------|---------|---------|
| 5 | 20 | 12.3% | 18.5% |
| 10 | 30 | 15.7% | 15.2% |
| 20 | 60 | 9.8% | 12.1% |

## 关键发现

1. **参数敏感**：不同参数组合收益差异巨大
2. **过度拟合**：在训练集上表现好的参数，在测试集上可能失效
3. **滑点影响**：实际交易中滑点会显著侵蚀收益

## 改进方向

- 加入成交量过滤
- 增加止损机制
- 结合波动率指标（ATR）动态调整仓位
