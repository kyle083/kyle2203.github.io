## 项目概述

一个开箱即用的 A 股多因子选股系统，整合了数据采集、因子计算、回测分析和可视化展示。

## 核心功能

### 1. 因子计算

- **价值因子**: PE、PB、PS、股息率
- **动量因子**: 20日/60日收益率、RSI
- **质量因子**: ROE、毛利率、负债率
- **成长因子**: 营收增长率、利润增长率

### 2. 回测引擎

```python
# 等权买入 Top 20 股票
portfolio = df.nlargest(20, 'composite_score')
returns = portfolio['next_return'].mean()
```

### 3. 可视化分析

使用 Streamlit 构建交互式 dashboard，展示：
- 因子 IC 值和衰减曲线
- 分层回测收益
- 多空组合净值走势
